Base class from which Node and Relationship classes inherit. Extends a rest.Resource by allowing additional URIs to be stored which represent both an Index and the entry within that Index which points to this resource. Additionally, provides property containment functionality.

Creates a representation of an indexable resource (Node or Relationship) identified by URI; optionally accepts further URIs representing both an Index for this resource type plus the specific entry within that Index.

Parameters:
  • uri - the URI identifying this resource
  • index_entry_uri - the URI of the entry in an Index pointing to this resource (optional)
  • index_uri - the URI of the Index containing the above entry (optional)
  • index - an index of RESTful URIs (optional)
  • http - httplib2.Http object to use for requests (optional)
  • user_name - the user name to use for authentication (optional)
  • password - the password to use for authentication (optional)
