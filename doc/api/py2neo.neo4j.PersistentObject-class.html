Class PersistentObject

source code

object --+
         |
        PersistentObject

A base object from which persistent objects may inherit. Unlike some persistence models, object attributes are updated "live" and therefore do not require an explicit save or load. Simple-typed attributes are mapped to Node properties where possible and attributes pointing to other PersistentObject instances are mapped to Relationships. No direct mapping to Relationship properties exists within this framework. All attributes begining with an underscore character are not mapped to the database and will be stored within the object instance as usual.

>>> from py2neo import neo4j
>>> gdb = neo4j.GraphDatabaseService("http://localhost:7474/db/data")
>>> class Country(neo4j.PersistentObject):
...     def __init__(self, node, name, population):
...             neo4j.PersistentObject.__init__(self, node)
...             self.name = name
...             self.population = population
...             self.currency = None
...             self.continent = None
...
>>> uk = Country(gdb.create_node(), "United Kingdom", 62698362)
